Welcome to Scgate's Documentation
========================================

`Scgate`_ (click  :download:`me <./_build/latex/simulocean.pdf>` to download user manual) is a web-based
deployment and visualization framework for coastal modeling and beyond.Scgate collects observational data,
schedules modeling codes for execution, manages data transfer, visualizes both observational and numerical results.
With all the information collected, Scgate can also provide direct validation and verification for models,
and generate high quality technical reports.
